# Calibration and Scoring

## Calibration

- Calibration session + user body profile are used to tune analysis expectations.
- Calibration data influences readiness and movement interpretation paths.

## Scoring

- Pose and motion signals flow into biomechanics and quality engines.
- Drill-specific metrics roll up into per-session score and issue summaries.
- Summary/recommendation generation converts technical outputs into coaching feedback.

## Imported Video Flow

- Imported clips use video frame sources and pose detection pipelines.
- Analysis/scoring flow is parallel in intent to live coaching, but source timing and orchestration differ.
