# CaliVision Studio (Local Package IO Foundation)

This folder contains the PR 2 local-first Studio prototype focused on package import/export compatibility.

## Run

```bash
cd studio
npm install
npm run dev
```

## Scope in this phase

- sample package loading
- local JSON package import
- package validation with structured issues
- package/drill/phase preview panels
- local JSON export

## Deferred

- pose editing canvas
- MediaPipe integration
- cloud persistence/auth/publishing
